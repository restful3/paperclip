### 5. 그림 1-1의 adapter 박스가 현재 패키지 지도와 맞지 않음

- **위치**: §1 그림 1-1 / `docs/report/assets/fig-02-monorepo.svg`
- **현재 본문 인용**: `그림 1-1은 패키지 지도와 의존 방향을 한 페이지로 보여 준다.`

- **근거**: 현재 그림은 `docs/report/assets/fig-02-monorepo.svg:36-58`에 8개 adapter만 표시한다(`claude-local`, `codex-local`, `cursor-local`, `gemini-local`, `opencode-local`, `pi-local`, `acpx-local`, `openclaw-gateway`). 실제 package형 adapter는 10개이고 `cursor-cloud`, `grok-local`이 빠져 있다. 또한 그림 `:96-98`은 `hermes-paperclip-adapter`를 external adapter plugin 예시로만 그리지만, 현재 upstream 코드에서는 `server/src/adapters/registry.ts:128-138`에서 정적 import되고 `:523`에서 built-in 배열에 포함된다.
