## 2. 남은 KB 표기 일부가 실제 파일 크기와 맞지 않음

**위치**  
`docs/report/05-server-api.md` §2 표 1 line 39, §3 코드 1 line 70·83

**현재 본문 인용**

> `agents.ts (28 KB)`  
> `environment-runtime.ts (44 KB)`

**문제 카테고리**  
정합성 — 엄격

**제안 수정안**  
KB 표기를 현재 파일 크기 기준으로 정리하세요.

> `plugins.ts` (94 KB), `plugin-ui-static.ts`

```text
│   ├── agents.ts (25 KB)

│   ├── environment-runtime.ts (46 KB)
```

> `plugins.ts`, `plugin-ui-static.ts`  
> `agents.ts`  
> `environment-runtime.ts`

## 3. 작업 공간/런타임 대표 엔드포인트가 라우트 그룹과 실제 경로를 혼동함

**위치**  
`docs/report/05-server-api.md` §2 표 1 line 38

**현재 본문 인용**  
> `environments.ts`, `execution-workspaces.ts` | 작업 공간/런타임 | `POST /api/projects/:pid/workspaces`

**문제 카테고리**  
정합성 — 엄격

**제안 수정안**  
대표 엔드포인트를 실제 파일 소속에 맞게 나누세요.

> `environments.ts`, `execution-workspaces.ts`, `projects.ts` | 환경·실행 작업 공간·프로젝트 workspaces | `GET/POST /api/companies/:companyId/environments`, `GET /api/companies/:companyId/execution-workspaces`, `GET/POST /api/projects/:id/workspaces`

짧게 유지하려면:

> `environments.ts`, `execution-workspaces.ts` | 환경/실행 workspace 조회·런타임 제어 | `GET/POST /api/companies/:companyId/environments`, `GET /api/companies/:companyId/execution-workspaces`, `POST /api/execution-workspaces/:id/runtime-services/:action`

그리고 project workspace 생성은 `projects.ts` 행 또는 별도 설명으로 보내는 편이 정확합니다.

**근거**  
`POST /api/projects/:id/workspaces`는 `environments.ts`나 `execution-workspaces.ts`가 아니라 `projects.ts`에 있습니다. 파라미터명도 `:pid`가 아니라 `:id`입니다.

```ts
// server/src/routes/projects.ts

...
258 router.post("/projects/:id/workspaces", validate(createProjectWorkspaceSchema), async (req, res) => {
```
