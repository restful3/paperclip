- **위치**: line 63
- **현재 본문 인용**: `표 1의 7가지 권한은 *영향 반경*으로 정렬되어 있다 — 위에서 아래로 갈수록 단일 에이전트 → 회사 → 이슈/트리 → 단건 → 예산 → 결정 → 문서 단위로 좁혀진다.`
- **문제 카테고리**: 품질
- **제안 수정안**: 표 순서를 유지한다면 `표 1의 7가지 권한은 실행 중단, 작업 통제, 예산, 결정, 산출물 동결까지 보드가 가진 hard 권한을 한데 모은 것이다.`처럼 정렬 주장만 빼세요. 영향 반경 정렬을 유지하려면 회사 일시정지를 에이전트 일시정지보다 위로 옮겨야 합니다.
